Welcome to the Kubefirst Documentation

What is Kubefirst?

Kubefirst is a cloud native platform for application delivery with automated infrastructure workflows. It has fine-grained user access with Vault secrets management, and support for GitHub or GitLab integrations and Argo. Kubefirst Pro gives you an incredible UI to manage your clusters, platform and applications.

Check out the Getting Started section for
  • An overview of Kubefirst and Kubefirst Pro capabilities
  • A high-level FAQ
  • Details on Getting Support
Supported installs

Kubefirst offers local installation with k3d and support for install with multiple clouds including AWS, Azure, Google, Civo, DigitalOcean and more.

Check out the Install section for detailed instructions in your cloud.

Features

Kubefirst functionality including your default repositories, our GitOps catalog, the Metaphor sample application, and adding your clusters.
