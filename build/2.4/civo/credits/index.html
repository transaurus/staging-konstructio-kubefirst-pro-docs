The Tools We Love​

The kubefirst platform uses some of the most popular open source projects in the world, and some closed sources ones (as little as possible). This page is dedicated to giving those projects credit for their incredible offerings.

  • Actions Runner Controller (Apache 2.0)
  • Argo CD (Apache 2.0)
  • Argo Workflows (Apache 2.0)
  • Atlantis (Apache 2.0)
  • cert-manager (Apache 2.0)
  • ChartMuseum (Apache 2.0)
  • Crossplane (Apache 2.0)
  • Docker Desktop (Docker Subscription Service Agreement)
  • External Secrets Operator (Apache 2.0)
  • GitHub Actions (GitHub license)
  • GitLab Runner (GitLab SaaS license)
  • HashiCorp Terraform (MPL 2.0)
  • HashiCorp Vault (MPL 2.0)
  • Ingress NGINX Controller (Apache 2.0)
  • k3d (MIT)
  • Kubernetes (Apache 2.0)
  • Kubernetes ExternalDNS (Apache 2.0)
  • MinIO (AGPL 3.0)
  • mkcert (BSD 3)
  • ngrok (ngrok license)
  • Reloader (Apache 2.0)
  • Traefik (MIT)
  • vcluster (Apache 2.0)
note

Please evaluate each licenses for third-party products used by kubefirst to validate if you can legally use them for your specific use case and legal status.
