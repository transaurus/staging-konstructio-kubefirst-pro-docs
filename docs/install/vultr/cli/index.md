Follow these steps to install Kubefirst with Vultr and the Kubefirst CLI.

- [Verify the prerequisites](../cli/vultr-cli-prereq.md) and install the Kubefirst CLI (with MacOS, Linux, or WSL).
- Complete the [install steps](../cli/vultr-cli-install.md) using the CLI
  
These installation steps are valid for both Kubefirst and Kubefirst Pro with access to the Kubefirst UI as a Free tier user and support the option to upgrade to either the Business or Enterprise tier.
