Follow these steps to install Kubefirst with Azure and the Kubefirst CLI.

- [Verify the prerequisites](../../azure/cli/azure-cli-prereq.md) and install the Kubefirst CLI ((with MacOS, Linux, or WSL).
- Complete the [install steps](../../azure/cli/azure-cli-install.md) using the Kubefirst CLI.

These installation steps are valid for both Kubefirst and Kubefirst Pro with access to the Kubefirst UI as a Free tier user and support the option to upgrade to the Business or Enterprise tier.
