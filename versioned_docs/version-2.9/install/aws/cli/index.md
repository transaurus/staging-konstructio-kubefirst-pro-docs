Follow these steps to install Kubefirst with AWS and the CLI

- [Verify the prerequisites](../cli/aws-cli-prereq.mdx) and install the Kubefirst CLI (with MacOS, Linux, or WSL).
- Complete [the install steps](../cli/aws-cli-install.mdx) using the Kubefirst CLI.

These installation steps are valid for both Kubefirst and Kubefirst Pro with access to the Kubefirst UI as a Free tier user and support the option to upgrade to either the Business or Enterprise tier.
