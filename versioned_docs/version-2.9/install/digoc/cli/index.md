Follow these steps to install Kubefirst with DigitalOcean and the Kubefirst CLI

- [Verify the prerequisites](../cli/do-cli-prereq.md) and install the Kubefirst CLI (with MacOS, Linux, or WSL).
- [Complete the install steps](../cli/do-cli-install.md)

These installation steps are valid for both Kubefirst and Kubefirst Pro with access to the Kubefirst UI as a Free tier user and support the option to upgrade to either the Business or Enterprise tier.

After installation check out the [Features section of our docs](../../../features/) to learn more about working with Kubefirst.
