Follow these steps to install Kubefirst with Akamai and the Kubefirst CLI.

- [Verify the prerequisites](../cli/akamai-cli-prereq.md) and install the Kubefirst CLI (with MacOS, Linux, or WSL).
- Complete the [install steps](../cli/akamai-cli-install.md) using the CLI (with macOS & Linux, Linux, or WSL)

These installation steps are valid for both Kubefirst and Kubefirst Pro with access to the Kubefirst UI as a Free tier user and support the option to upgrade to either the Business or Enterprise tier.
