Follow these steps to install Kubefirst with Akamai and the Kubefirst UI.

- Verify the [required prerequisites](../ui/akamai-ui-prereq.md)
- Complete the [install steps](../ui/akamai-ui-install.md) using Helm or the Kubefirst command depending on your configuration

These installation steps are valid for both Kubefirst and Kubefirst Pro with access to the Kubefirst UI as a Free tier user and support the option to upgrade to either the Business or Enterprise tier.
