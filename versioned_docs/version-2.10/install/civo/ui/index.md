:::tip
This installation method is recommended for users that **do not have an
existing Kubernetes cluster** to use for installation. If you have an existing cluster
check out our instructions [for installation using Civo and Helm](../helm/index.md).
:::

Follow these steps to install Kubefirst with Civo and the Kubefirst UI.

- [Verify the required prerequisites](civo-ui-prereq.mdx)
- [Complete the install steps](civo-ui-install.mdx)

These installation steps are valid for both Kubefirst and Kubefirst Pro with access to the Kubefirst UI as a Free tier user and support the option to
upgrade to either the Business or Enterprise tier.
